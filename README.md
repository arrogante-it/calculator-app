# calculator-app

Calculator for basic operation +/*-.
 
Tested.

Prerequisite:
-
- JDK 11 or higher
- Maven 4.0.0

Program main functional:
-
- add:
`/api/v1/calculator/add?number1=6&number2=3`

- subtract:
`/api/v1/calculator/add?number1=6&number2=3`

- multiply:
`/api/v1/calculator/add?number1=6&number2=3`

- divide:
`/api/v1/calculator/add?number1=6&number2=3`
